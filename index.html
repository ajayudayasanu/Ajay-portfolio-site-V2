Web Development

Frontend Web development is one of my favourite fields in technology. I use React and angular as my framework for web development. Bootstrap and Semantic UI are my favourite CSS libraries. I do styling in SCSS. I love Netlify and AWS for hosting the application. I code the business logic in Javascript/Typescript.

React Native Android/Ios mobile application development

Native mobile application development using React Native is impressive. we can develop an application for both Android and IOS from the same code base. also, we will be able to share the code from the web application as well. with the introduction of react-native, we can have a mono repository for both web and mobile application

What I DO
